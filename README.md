##新闻客户端

####基本信息
* 开发环境：Android Studio 1.4
* 运行环境：Android 5.0
* 测试机型：魅族MX3
* 使用第三方框架：slidingmenu、ViewPagerIndicator、XUtils3、极光推送、社会化分享等

####注意事项
* 服务端：使用Tomcat8，JSON数据是从网络上找的
* 使用方法：
    * 运行时如果没有内容，请将文件内的服务端JSON文件放置服务器
    * 代码GlobalContants类中封装了服务器的地址，请根据自己的服务器网站，自行更改
